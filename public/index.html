Persus's Shield

A shiny recepie to serve resized images on-the-fly

 
  • Source image URL is passed as param.
  • URL is checked against a regex to avoid public use.
  • Image is resized using the fastimage_resize gem.
  • Resized image is stored on Amazon's S3 for good (configurable).
  • Served using a redirect to a CloudFront URL.
  • Redis is used for caching keys of created images.
